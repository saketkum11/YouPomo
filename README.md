# Getting Started with Create React App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

URL - https://youpomo.netlify.app/

Features 
- create tasks 
- update tasks
- delete tasks
- pomodoro clock
- stop, pause and reset clock

Tools

- react toastify 
- font awesome
